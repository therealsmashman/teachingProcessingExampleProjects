Teaching Processing Example Projects

A collection of simple projects used to teach programming in Processing,

Download .zip Download .tar.gz View on GitHub

This is a set of small stand alone programs in processing that written by beginner programmers.

Each program is designed to be interesting and cover specific sections of programming.

Dice

Displays a simple dice, that changes number automatically on a timer. Includes some shaking animation.

Covers:

  • Random
  • Switch statements
  • Manipulating the framerate of the animation by changing the frame rate of the draw function.

Key concepts:

Getting the number to display on the dice:

number = (int)random(1, 7);

Drawing different dice faces depending on the number to be shown on the dice:

switch(number) {
  case 1: 
    ellipse(diceCenterHos, diceCenterVert, 20, 20);
    break;
  case 2:
    ellipse(diceCenterHos -20, diceCenterVert, 20, 20);
    ellipse(diceCenterHos +20, diceCenterVert, 20, 20);
    break;
    ...
}

Roll with button

An extension of the dice program, for students who have finished that early. Uses user input to trigger a new roll of the dice.

Covers:

  • User input
  • Button section (collision detection on a circle).

Key concepts:

Checking whether the user has clicked on a button.

void mouseReleased() {
  if(dist(mouseX, mouseY, buttonCenter, buttonCenter)< 50){
    roll();
  }
}

Read In From Keyboard

Reads data in from the keyboard, displays it on the screen

The base version only deals with charecters from 'a' - 'z', when students have finished that they can move onto numbers, capitals and special charecters such as backspace and enter.

Covers:

  • Keyboad input
  • Basic string manipulation
  • The AND condition
  • Displaying text to the screen
  • Can cover special charecters.

Key concepts:

Adding pressed keys to a string:

void keyPressed() {
  if (key >= 'a' && key <= 'z') {
    keyboardInput += key;
  } 
  ...
  }

Quotes Loader

Loads quotes from a text file.

Loads each line from a list of quotes, stores them in an array. Then shows a random quote each time the mouse is clicked.

Covers:

  • File I/O
  • String arrays
  • For Loops
  • Random

Key concepts:

Loading strings line by line from a file:

quotes  = loadStrings("quotes.txt");

Number guesser

A simple guessing game

The user has to guess a number between 0 and 20, getting hints of higher or lower on incorrect guesses.

Covers:

  • Simple logic
  • Draw text

Converting input strings into numbers (ints) for comparison:

void keyPressed() {
  if (key >= '0' && key <= '9') {
    nextGuess = nextGuess *10 + ((int)key -48) ;
  }

  }

Cars Class Example

A simple introduction to classes, using a "Car" class.

We create two car objects and then make them move at different speeds.

Covers:

  • Classes
  • Objects

Key concepts:

Creating a new class

class car{
....
public car(int inStartHozPos, int inStartVertPoz){
    carWidth = 40;
    carHeight = 20;
    speed = 2;
    carHozPoz = inStartHozPos;
    carVertPoz = inStartVertPoz;
  }
  ...
  }

Creating new objects of that class, and each being a seperate object with different private variables

//initialize the new car objects.
  //this calls the "car" constructor
  car1 = new car(20,20);
  car2 = new car(20,100);
  //now we call some of the methods on the car objects.
  //this will change some of the internal values - like carHeight and speed
  car1.setSpeed(4);
  car2.setCarSize(60,40);

Movie Googler

User an api to retrieve information about films and tv shows.

Get a name from the user, then searches on that name, and if a result us found, returns data and a poster image for that result.

Uses the omdbapi api.

Uses XML rather tahn JSON as its slightly more intuative to understand and Processing supports XML parsing.

Covers:

  • APIs: as little or as much as you want
  • XML: what it is, structure and parsing
  • String arrays
  • For loops
  • Button using collition detection
  • Null checking
  • Keyboard input
  • Images

Key concepts:

Calling an API

String queryStart = "http://www.omdbapi.com/?r=xml&t=";
...
loadStrings(queryStart + movieNameForApi)[0];

Parsing XML result, and null checking

int line = 20;
  for (int j = 0; j< attributesToShow.length; j=j+1) {
    if (movieXML.hasAttribute(attributesToShow[j])) {
      drawText(attributesToShow[j].toUpperCase() + " : "+movieXML.getString(attributesToShow[j]), line, 15);
      line=line + 15;
    }
  }

Loading image from the web

PImage poster;
...
if (movieXML.hasAttribute("poster")) {    
    poster = loadImage(movieXML.getString("poster"));
    poster.resize(200, 300);
    image(poster, width-200, 0);
  }

Simple Music Player

Very simple app that loads a MP3 file and then plays it using the external "Minim" library that comes with processing.

  • Loading files
  • Audio players

Loading file into a local object and then calling a method on that.

myAudioPlayer = minim.loadFile("cuteShort.mp3");
...
myAudioPlayer.play();

Simple Music Player With Keyboard Controls

An extenstion of the Simple Music Player.

Uses input from the keyboard to pause and play the music.

Covers:

  • Extyending an exiting program
  • Keyboard input
  • Audio players
  • String manipulation - multiline strings using the newline charecters "\n"

Key concepts:

Calling pause and play depending on keys pressed.

if (keyPressed) {
    if (key == 'p') {
      myAudioPlayer.play();
    }
    if (key == 'a') {
      myAudioPlayer.pause();
    }
  }

Multiline strings

drawText("Controls: \n"+
            "'p' to play \n"+
            "'a' to pause \n" +
            "the music");
