Teaching Processing Example Projects

A collection of simple projects used to teach programming in Processing,

Download .zip Download .tar.gz View on GitHub

This is a set of small stand alone programs in processing that written by beginner programmers.

Each program is designed to be interesting and cover specific sections of programming.

dice

Displays a simple dice, that changes number automatically on a timer. Includes some shaking animation.

Covers:

  • Random
  • Switch statements
  • Manipulating the framerate of the animation by changing the frame rate of the draw function.

Key concepts:

Getting the number to display on the dice:

number = (int)random(1, 7);

Drawing different dice faces depending on the number to be shown on the dice:

switch(number) {
  case 1: 
    ellipse(diceCenterHos, diceCenterVert, 20, 20);
    break;
  case 2:
    ellipse(diceCenterHos -20, diceCenterVert, 20, 20);
    ellipse(diceCenterHos +20, diceCenterVert, 20, 20);
    break;
    .....
}

roll with button

An extension of the dice program, for students who have finished that early. Uses user input to trigger a new roll of the dice.

Covers:

  • User input
  • Button section (collision detection on a circle).

Key concepts:

Checking whether the user has clicked on a button.

void mouseReleased() {
  if(dist(mouseX, mouseY, buttonCenter, buttonCenter)< 50){
    roll();
  }
}

Read In From Keyboard

Reads data in from the keyboard, displays it on the screen

The base version only deals with charecters from 'a' - 'z', when students have finished that they can move onto numbers, capitals and special charecters such as backspace and enter.

Covers:

  • Keyboad input
  • Basic string manipulation
  • The AND condition
  • Displaying text to the screen
  • Can cover special charecters.

Key concepts:

Adding pressed keys to a string:

void keyPressed() {
  if (key >= 'a' && key <= 'z') {
    keyboardInput += key;
  } 
  ....
  }

Quotes Loader

Loads quotes from a text file.

Loads each line from a list of quotes, stores them in an array. Then shows a random quote each time the mouse is clicked.

Covers:

  • File I/O
  • String arrays
  • For Loops
  • Random

Key concepts:

Loading strings line by line from a file:

quotes  = loadStrings("quotes.txt");

Number guesser

A simple guessing game

The user has to guess a number between 0 and 20, getting hints of higher or lower on incorrect guesses.

Covers:

  • Random
  • Simple logic
  • Draw text
  • A modified version of the keyboaard input method

Key concepts:

Converting input strings into numbers (ints) for comparison:

void keyPressed() {
  if (key >= '0' && key <= '9') {
    nextGuess = nextGuess *10 + ((int)key -48) ;
  } 
  ....
  }
